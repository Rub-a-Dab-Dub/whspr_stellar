================================================================================

   docker-compose up -d
   (or: sudo docker-compose up -d)

2. Run Migrations:
   npm run migration:run

3. Start Server:
   npm run start:dev

4. Access API:
   - API Info: http://localhost:3001/api
   - Swagger: http://localhost:3001/api/docs
   - Health: http://localhost:3001/api/health
